快速开始 ​

在开始之前，您首先应该：

  • 熟悉 Java 环境配置及其开发
  • 熟悉 关系型 数据库，比如 Postgres
  • 熟悉 Spring Boot
  • 熟悉 网络游戏服务器框架ioGame
  • 熟悉 网络游戏服务器框架Mybatis-Flex
  • 熟悉 Java 构建工具，比如 Maven
